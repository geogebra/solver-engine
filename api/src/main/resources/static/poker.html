Input syntax
  • recurring decimals: \(10.3\overline{401}\) is 10.3[401],
  • arithmetic operators \(+, -, \times, \div\) are +, -, *, :,
  • \(x^y\) is [x ^ y],
  • \(\frac{x}{y}\) is [x / y],
  • \(2\frac{1}{3}\) is [2 1/3],
  • \(\sqrt{x}\) is sqrt[x],
  • \(\sqrt[3]{12}\) is root[12, 3].

Example: \(\frac{1}{\sqrt{1 - x^2}}\) is [1 / sqrt[1 - [x^2]]].
